\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\item
  Let \(f(x) = c\left[1 - (x - 1)^2\right]\). Find the value of \(c\)
  and support set for which \(f\) is a PDF.
\item
  Intuitively, a log transformation should reduce the skewness of a
  distribution. The skewness of a random variable is defined as the
  third standardized moment, i.e., the expectation
  \(\mathbb{E}\left[\left(\frac{X - \mu}{\sigma}\right)^3\right] = \frac{\mu_3 - 3\mu_1\sigma^2 - \mu_1^3}{\sigma^3}\),
  where \(\mu_k\) denotes the \(k\)th moment and \(\sigma\) denotes the
  square root of the variance, i.e.,
  \(\sigma = \sqrt{\mu_2 - \mu_1^2}\). Let \(\log(X) \sim N(0, 1)\);
  show that the skewness of \(X\) is larger than the skewness of
  \(\log(X)\).
\item
  Let \(X \sim \text{exponential}(\beta)\) and \(c > 0\) and define \[
   Y = \begin{cases}
     1 &,\; X > c \\
     0 &,\; X \leq c
   \end{cases}
   \] Find the distribution of \(Y\) and its mean and variance.
\end{enumerate}
